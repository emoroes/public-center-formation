# Поиск и оценка нового общественного центра в административном районе

Интерактивное Streamlit-приложение для поиска и оценки потенциальных общественных центров в административных районах Санкт-Петербурга. Работа использует открытые пространственные данные OpenStreetMap, рассчитывает центральные места по жилой застройке и помогает оценить выбранный объект по нескольким городским критериям.

**Автор:** Мороз Евгений Юрьевич 

**Тема работы:** Разработка метода формирования общественных центров в административных районах города

## Краткое описание

Приложение позволяет выбрать район Санкт-Петербурга, задать время пешеходной доступности и число центров, после чего строит карту с расчетными центральными местами. Для выбранного здания на карте дополнительно рассчитываются оценки:

- масштабность объекта по площади застройки;
- социально-культурная активность окружения;
- центричность относительно расчетных зон;
- транспортная доступность.

## Исходные данные

Основной источник данных - OpenStreetMap, загружаемый через библиотеку OSMnx:

- границы административных районов Санкт-Петербурга;
- контуры зданий и их OSM-теги;
- пешеходный граф улично-дорожной сети;
- объекты социально-культурной активности;
- остановки общественного транспорта и входы метро.

В папке `data/` добавлены подготовленные справочные таблицы:

- `districts.csv` - список районов и строковые запросы OSMnx;
- `district_population.csv` - численность населения районов, используемая для рекомендации количества центров;
- `data/README.md` - описание состава данных и логики их использования.

## Метод

1. Для выбранного района загружаются граница, здания и пешеходный граф.
2. Из зданий отбирается жилой фонд: `residential`, `apartments`, `house`, `detached`, `dormitory` и близкие типы.
3. Для каждого жилого здания рассчитывается вес как приближение численности жителей: площадь застройки умножается на этажность, если она доступна в OSM.
4. Центральные места находятся методом Weighted K-means по центроидам жилых зданий.
5. Пешеходная доступность оценивается либо приближенно по прямому расстоянию, либо по графу пешеходной сети.
6. Для выбранного объекта рассчитываются частные оценки и итоговый балл.

## Примечания

Результаты зависят от полноты и актуальности тегов OpenStreetMap. Полученные центры следует рассматривать как аналитическую поддержку принятия решений, а не как окончательное градостроительное заключение.
