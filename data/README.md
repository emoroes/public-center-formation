# Исходные данные

Данные проекта состоят из двух частей: подготовленных справочников в репозитории и динамически загружаемых пространственных данных OpenStreetMap.

## Подготовленные файлы

`districts.csv` содержит список административных районов Санкт-Петербурга и запросы, которые используются для загрузки геоданных через OSMnx.

`district_population.csv` содержит численность населения районов. Эти значения используются в приложении для расчета рекомендуемого числа общественных центров по правилу: один центр на 200 000 человек.

Формат файлов - CSV с кодировкой UTF-8.

## Динамически загружаемые данные

Через OpenStreetMap загружаются:

- границы районов;
- здания и атрибуты зданий;
- пешеходный граф;
- точки интереса для социально-культурной активности;
- объекты общественного транспорта.

Приложение кэширует загруженные данные локально для ускорения повторных расчетов. Кэш не является исходными данными исследования и не включается в репозиторий.

## Ограничения

OpenStreetMap наполняется сообществом, поэтому часть атрибутов может отсутствовать или быть неоднородной. В методе используются устойчивые признаки: тип здания, площадь геометрии, этажность, транспортные и социально-культурные теги.
